# streamlit-image-filter-app
Developed a Streamlit-based Image Filter Web App using Python. The application allows users to upload images and apply filters such as Grayscale, Blur, Contour, and Detail using the PIL library. This project demonstrates basic image processing, interactive UI design, and Python web app development.
## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   streamlit run image.py
